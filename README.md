# 📋 Gerenciador de Tarefas em Python

Projeto desenvolvido para praticar lógica de programação e os conceitos fundamentais da linguagem Python utilizando apenas o terminal.

## 📌 Sobre o projeto

Este programa permite ao usuário gerenciar uma lista de tarefas diretamente pelo terminal.

As tarefas ficam armazenadas em uma lista durante a execução do programa e podem ser salvas em um arquivo de texto (`tarefas.txt`) ao finalizar.

---

## ✨ Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Excluir tarefas
- Salvar as tarefas em um arquivo `.txt`

---

## 🛠️ Tecnologias utilizadas

- Python 3

## ▶️ Como executar

1. Clone este repositório:

```bash
git clone https://github.com/Edelly-Nogueira/gerenciador-de-tarefas-python.git
```

2. Entre na pasta do projeto:

```bash
cd gerenciador-de-tarefas-python
```

3. Execute o programa:

```bash
python main.py
```
